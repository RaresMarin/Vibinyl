# 💿 Vibinyl

 Vibinyl is a vinyl shop! Buy your favorite albums!

## ❤️‍🔥 Features
 
- Add code for points 
- Buy the albums
- Log in/Log out
- Spining vinyl animation

## 💽 Technologies Used

- JavaScript
- CSS
- HTML5


## 📀 How to Use

- Launch the app to buy vinyls;
- Click the buttons to see all the pages;
